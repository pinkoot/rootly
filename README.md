###  Цель проекта rootly
![[Pasted image 20260225143247.png]]

Разработать веб-платформу для проведения соревнований формата CTF (Capture The Flag) с использованием фреймворка Django со реализацией следующих задач:
* управления заданиями
* автоматической проверки флагов
* ведения рейтинга участников
### Функциональныее требования
##### Пользовательская система
* Регистрация пользователя
* Авторизация / Logout
* Профиль пользователя
* Восстановление пароля
##### Управление заданиями
* Создание заданий
* Указание категории
* Динамическая система очков
* Возможность прикрепления файлов к заданиям
* Возможность указания ссылки на внешний сервис
##### Проверка флагов
* Отправка флага игроком
* Проверка системой:
     * Совпадение строки
     * Учет регистра
##### Таблица лидеров
* Общий рейтинг
* Обновление в реальном времени
##### Безопасноть
Реализовать защиту от базовыз веб-уязвимостей
